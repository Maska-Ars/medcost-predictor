# Предсказатель стоимости страховки

Веб-приложение на базе машинного обучения для оценки стоимости медицинской страховки на основе персональных данных пациента.

## Описание

Приложение использует обученную модель **CatBoostRegressor** для предсказания стоимости страховки. Модель учитывает следующие факторы:
- Возраст
- Пол
- Индекс массы тела (ИМТ)
- Количество детей
- Статус курения
- Регион проживания

## Установка и запуск

### Требования
- Python 3.12+
- Установленные зависимости (см. `requirements.txt`)

### Установка

```bash
git clone <https://github.com/Maska-Ars/medcost-predictor/tree/master>
cd <medcost-predictor>
pip install -r requirements.txt
python app.py
```

### Использование
Введите данные пациента в форму слева:
- Возраст (от 0 до 200 лет)
- Пол (Male/Female)
- ИМТ (от 10 до 100)
- Количество детей (от 0 до 100)
- Статус курения (Да/Нет)
- Регион (northeast, northwest, southeast, southwest)

Нажмите "Предсказать стоимость" для получения оценки

Используйте примеры внизу страницы для быстрого тестирования

Очистите поля кнопкой "Очистить поля" при необходимости
